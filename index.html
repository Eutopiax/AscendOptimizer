Abstract

NPU Optimization LLM Agent

AscendC operator optimization on Huawei Ascend NPUs faces a two-fold knowledge bottleneck: unlike the CUDA ecosystem, there are few public reference implementations, and performance depends on a coupled two-part artifact‚Äîa host-side tiling program and a kernel program.

We present AscendOptimizer, an episodic agent that bootstraps expertise by turning execution into experience. On the host side, it performs profiling-in-the-loop evolutionary search to discover valid and high-performing tiling configurations. On the kernel side, it mines transferable motifs by rewinding optimized kernels‚Äîsystematically de-optimizing them to synthesize instructive "bad-to-good" trajectories‚Äîand distills these into a retrievable experience bank.

Key Results: On 133 AscendC operators, AscendOptimizer achieves 1.34√ó GM speedup over baseline, with 51.88% outperforming references.

Alternating Optimization Framework

AscendOptimizer alternates between optimizing tiling ùíØ and kernel Œ∫ in two stages:

Stage I: Evolutionary-Guided Program Search

  • Models tiling as program search
  • LLM-driven evolutionary search
  • Hardware-in-the-loop feedback

Stage II: Optimization-Rewind Experience

  • Rewinds optimizations in seeds
  • Constructs pattern library
  • Retrieval-augmented rewriting
