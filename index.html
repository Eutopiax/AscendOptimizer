Abstract

AscendC operator optimization on Huawei Ascend neural processing units (NPUs) faces a two-fold knowledge bottleneck: a lack of public reference implementations and the need to tightly couple host-side tiling with device-side kernel scheduling [cite: 10].

We present AscendOptimizer, an episodic agent that bootstraps this missing expertise by turning execution into experience [cite: 11]. On the host side, it performs profiling-in-the-loop evolutionary search to discover high-performing tiling configurations [cite: 12]. On the kernel side, it mines transferable optimization motifs by rewinding optimized kernels to synthesize "bad-to-good" trajectories [cite: 13]. On a benchmark of 133 real AscendC operators, AscendOptimizer achieves a 1.34x geometric-mean speedup over the open-source baseline, outperforming strong agent and search baselines [cite: 15].

Alternating Optimization Framework [cite: 132]

AscendOptimizer optimizes the host-side tiling function and the device-side kernel code in two stages [cite: 149]:

  • Stage I: Evolutionary-Guided Program Search. Modeled as a program search problem, it leverages LLMs to perform evolutionary search, using hardware-in-the-loop execution feedback to implicitly learn hardware constraints [cite: 152].
  • Stage II: Optimization-Rewind based Experience Bootstrapping. Constructs a structured optimization pattern library via deliberate de-optimization ("Optimization Rewind"), transforming infinite code search into finite expert experience retrieval and application [cite: 154].
